## Getting Started :
### Pre-requisites
1.  Ruby
2.  Postgres
3.  Git

### Installation :
1.  Clone this repository into a folder
2.  Run `bundle install` inside the cloned repository


### Used Versions :

* Ruby version : ruby-2.6.3
* FriendlyId version : 5.2.4
* Configuration :
    DB : Postgresql
